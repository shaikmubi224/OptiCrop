About OptiCrop

Project Overview

OptiCrop is an AI-powered smart agricultural recommendation system developed using Machine Learning. The system analyzes soil nutrients (Nitrogen, Phosphorous and Potassium), temperature, humidity, pH and rainfall to recommend the most suitable crop.


Machine Learning Models

• Logistic Regression
• Decision Tree
• Random Forest
• K-Nearest Neighbors (KNN)
• K-Means Clustering
